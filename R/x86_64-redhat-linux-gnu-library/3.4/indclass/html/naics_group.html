naics_group {indclass}R Documentation

Convert full NAICS codes to 2, 3, 4, or 5 digit codes

Description

Converts NAICS industry classifications into their broader categories by shortening the 6 digit code to 2, 3, 4, or 5 digits

Usage

naics_group(NAICS, digit)

Arguments

NAICS

A numeric vector of NAICS codes

digit

How many digits to shorten the NAICS code by

Value

A numeric vector of shortened NAICS industry codes

Examples

x <- c(800000,205000,190000)
naics_group(x, 3)
