sic_to_ff38 {indclass}R Documentation

Convert SIC codes to Fama French 38 industry codes

Description

Converts SIC codes to their corresponding industry code using the Fama-French 38 industry portfolio classifications

Usage

sic_to_ff38(SIC)

Arguments

SIC

A numeric vector of SIC codes

Value

A numeric vector of Fama-French 38 industry portfolio codes

Examples

x <- c(800,2000,4537)
sic_to_ff38(x)
