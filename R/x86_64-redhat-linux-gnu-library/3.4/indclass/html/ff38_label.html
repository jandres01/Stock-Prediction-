ff38_label {indclass}R Documentation

Convert Fama-French 38 industry codes to labels

Description

Converts a vector of Fama-French industry codes to the appropriate labels

Usage

ff38_label(ff38)

Arguments

ff38

A numeric vector of fama-french industry codes

Value

A vector of Fama-French 38 industry portfolio labels

Examples

x <- c(1,14,37)
ff38_label(x)
